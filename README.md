## Sorting Algorithms:
 <ul>
	 <li>Bubble Sort Algorithm
	 	<ul>
	 		<li> <b>O(1)</b> in base case </li>
	  		<li> <b>O(n^2)</b> in worst case </li>
	 	</ul> 
	 </li>
	 <li>Merge Sort Algorithm<br>
	 	<ul>
	 		<li> <b>O(1)</b> in base case </li>
	  		<li> <b>O(log n)</b> in worst case </li>
	 	</ul> 	
	 </li>
 </ul>
